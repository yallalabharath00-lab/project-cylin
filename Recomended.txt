git commands
 1.git init - to intialize git
 2.git add . -to update all live changes
 3.git commit -m "your change" - to update the changes as a commit
 4.git push origin main - to push the commits and changes to the main branch (or) github
 5.git pull <your github project link> - to pull the project to your repo
 6.git clone <repo link> - to clone the repo from github
 7.git log - to see the commits
 8.git branch - to check the branch 
 9.git switch <branch name> - to switch branches
